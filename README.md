# UlumKampusFS

This is a comprehensive educational management system for Ulum Kampus. The system includes features for academic calendar management, faculty and major administration, course management, grade tracking, and user role management.

## Features

- Academic Calendar Events API
- Faculty and Major Management
- Course and Assignment Management
- Grade Tracking and Analytics
- User Role Management (Admin, Faculty, Dosen, Prodi, Student)

## Architecture

The system is built with a Laravel backend API and follows RESTful API principles for all endpoints.

Detailed API specifications can be found in the `docs/` directory.

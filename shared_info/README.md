# Shared Information

This directory contains shared documentation and specifications for the UlumKampus project.

## Directory Structure

- `docs/` - Project documentation
- `specs/` - Technical specifications
- `api/` - API documentation and contracts
- `components/` - Component specifications
- `meetings/` - Meeting notes and decisions

## Purpose

This folder serves as a centralized location for information that needs to be shared between different parts of the project and with AI tools working on the codebase.

# AI Context Directory

This directory contains information designed to provide context for AI tools working on the UlumKampus project.

## Directory Structure

- `docs/` - Project documentation for AI reference
- `specs/` - Technical specifications
- `api/` - API documentation and contracts
- `components/` - Component specifications and requirements
- `meetings/` - Meeting notes and architectural decisions

## Purpose

This folder serves as a centralized location for information that provides context to AI tools about the project structure, requirements, and decisions. The underscore prefix makes this directory easily recognizable to AI tools as a source of project context.
